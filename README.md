## 📖 Sobre o Projeto

O **Use Mais Energia Solar** é um site educativo desenvolvido como trabalho de faculdade com o objetivo de conscientizar as pessoas sobre os benefícios da energia solar.  
Acreditamos que informação de qualidade é o primeiro passo para uma transição energética mais limpa e acessível.

O projeto reúne conteúdo claro e visualmente atraente para explicar como a energia solar funciona, seus impactos positivos no meio ambiente e na economia, além de incentivar sua adoção em residências e empresas.

## ✨ Funcionalidades

- **Conteúdo educativo** – textos explicativos sobre energia solar de forma simples e didática.
- **Design responsivo** – adaptado para computadores, tablets e smartphones.
- **Experiência agradável** – interface moderna e intuitiva, pensada para engajar o visitante.
- **Hospedagem na Vercel** – disponível 24/7 para consulta e compartilhamento.

## 🛠️ Tecnologias Utilizadas

- [Next.js](https://nextjs.org/) – framework React para renderização e performance.
- [React](https://react.dev/) – biblioteca para construção da interface.
- [TypeScript](https://www.typescriptlang.org/) – para um código mais seguro e organizado.
- [Tailwind CSS](https://tailwindcss.com/) – estilização rápida e responsiva.
